# mss-boot-admin-antd
mss-boot-admin前端项目，基于antdpro开发

## feature
- [x] 通过模板生成代码
- [x] github oauth2登录
- [ ] 定时任务管理
- [ ] 角色管理
- [ ] 角色授权
- [ ] 用户管理
- [ ] 菜单管理

### Start project

```bash
npm start:alpha
```

### Build project

```bash
npm run build:alpha
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
